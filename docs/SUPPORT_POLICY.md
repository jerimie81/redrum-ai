# Support Policy

## Bug Reports
Please file bug reports on the internal issue tracker. Include the output of `redrum-ai bug-report`.

## Data Recovery
We do not hold remote backups of your `memory.db`. If local corruption occurs, follow the instructions in `RUNBOOKS.md`.

## Model Runtime Issues
Support for local models (e.g., Ollama) is limited to the interface. We cannot troubleshoot GPU driver failures or out-of-memory errors on the host machine beyond logging the failure.
