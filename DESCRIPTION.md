# redrum-ai Description

`redrum-ai` is a personal AI partner project focused on building a durable working relationship between Redrum and a local LLM.

## Authority

`AGENT.md` and `manifest.json` are the highest-priority project directives for this companion. They must be followed expressly, and this description defers to them whenever there is any conflict.

## Summary

The goal is to evolve the current prototype into a practical assistant that can learn from repeated use, remember useful context, plan work, and help complete tasks across coding, operations, research, organization, and day-to-day execution.

This project is centered on partnership, not novelty. The assistant should become a reliable collaborator that supports the user with continuity, judgment, memory, and safe action.

## Current State

The project currently has:

- A SQLite-backed memory foundation
- A basic Python assistant script
- An Ollama-based local model workflow
- A CLI wrapper for invocation
- A project progress document with a staged roadmap

It is still early-stage and needs stronger packaging, safer tools, better retrieval, structured memory, testing, and a clearer autonomous workflow before it can function as a true companion.

## Target State

The end goal is a usable local assistant that can:

- Work with the user across many kinds of tasks
- Learn preferences and recurring workflows
- Keep track of projects and follow-through
- Plan work into actionable steps
- Use tools safely with approval boundaries
- Explain its actions clearly
- Improve through feedback and experience

## Success Criteria

The project should be considered on track when the assistant can:

- Understand a request with minimal back-and-forth
- Retain relevant user preferences and project context
- Recommend the next best step
- Execute approved actions safely
- Recover from errors without losing progress
- Provide useful status updates and summaries
- Support a real working partnership instead of one-off chat

## Intended Audience

This description is intended for the user and for any AI web app or project director that needs a concise statement of purpose, scope, and desired outcomes.
